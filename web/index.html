          # css几种居中方式




           


































          水平垂直居中：
定位和需要定位的元素的margin减去宽高的一半
img{
            width: 100px;
            height: 150px;
            position: absolute;
            top: 50%;
            left: 50%;
            margin-top: -75px;
            margin-left: -50px;
   }

定位和margin:auto
img{
            width: 100px;
            height: 100px;
            position: absolute;
            top: 0;
            left: 0;
            right: 0;
            bottom: 0;
            margin: auto;
   }

利用tranform实现的水平垂直居中 
img{
        width: 100px;
        height: 100px;
        position: absolute;
        top: 50%;
        left: 50%;
        transform: translate(-50%,-50%);
   }  
   
  浮动,定位,可以解决外边距合并的问题 固定定位,绝对对位 
  以前是用padding,boreder,overflow:hidden

          # css 新属性转载来自鑫空间❤

          • 2019-7-23
          • 1.filter: blur(5px)滤镜模糊 //你只需要给图片一个这个属性 px越大越模糊
          • 2.filter: brightness(1.4) //滤镜亮度
          • 3.filter: contrast(2); //滤镜对比度
          • 4.filter: drop-shadow(4px 4px 8px blue); //滤镜-投影
          • 5.filter: grayscale(50%); //滤镜-灰度
          • 6.filter: hue-rotate(90deg); //滤镜-色调变化
