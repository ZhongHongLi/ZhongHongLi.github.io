          # DOM 🔍

          (Document Obeject Model)简称DOM是BOM下面的一个子集.是 W3C 组织推荐的处理可扩展标记语言（HTML 或者XML）的标准编程接口。

          虽然现在框架都是肆意流行提倡减少dom操作,但是一些方法还是需要去复习

          # 获取元素 ⌛

          document.getElementById(id)

          document.getElementsByTagName('标签名')

          document.getElemntsByClassName('类名'

          document.querySelector('选择器')

          document.querySelectorAll('选择器')

          document.body获取body

          document.documentElement获取html

          # 事件概述 🛢

          JavaScript 使我们有能力创建动态页面，而事件是可以被 JavaScript 侦测到的行为。

          简单理解为--响应机制

          # 事件三要素 📌

          事件源=>事件类型=>事件处理程序

          # 常见鼠标事件 💡

          鼠标事件 触发条件
          onclick 鼠标点击左键触发
          onmouseover 鼠标经过触发
          onmouseout 鼠标离开触发
          onfocus 获取鼠标焦点触发
          onblur 失去鼠标焦点触发
          onmousemove 鼠标移动触发
          onmouseup 鼠标弹起触发
          onmousedown 鼠标按下触发

          # 改变元素内容 ⚗

          element.innerText:这个是针对文本

          element.innerHTML:这个可以识别html标签

          innerText和innerHTML的区别

          • 获取内容时的区别： innerText会去除空格和换行，而innerHTML会保留空格和换行

          -设置内容时的区别： innerText不会识别html，而innerHTML会识别

          # 常用元素的属性的操作

          • 1.innerText,innerHTML 改变元素的内容
          • 2.src,href
          • 3.id,alt,title

          # 表单元素的属性操作

          利用DOM可以操作如下表单元素的属性:

          type,value,checked,selected,disabled

          # 自定义属性操作

          1.getAttribute('属性');

          2.setAttribute('属性','值');

          3.removeAttribute('属性');

          事件对象属性方法 说明
          e.target 返回触发事件的对象
          e.srcElement 返回触发事件对象
          e.type 返回事件类型
          e.cancelBubble 该属性阻止冒泡 以前的写法

          # e.target和this的区别

          • this是事件绑定的元素 (绑定这个事件处理函数的元素)
          • e.target是事件触发的元素 正常情况下target和this是一致的 但是有一种情况不同，就是事件冒泡时(父子元素有相同的时间,单击子元素,父元素的事件处理函数也会被执行)； 这个时候this指向的是父元素,因为它是绑定事件的元素对象 而target指向的事子元素,因为它触发的事具体的元素对象

          鼠标事件对象 说明
          e.clientX 返回鼠标相对于浏览器窗口可视区的X坐标
          e.clientY 返回鼠标相对于浏览器窗口可视区的Y坐标

          e.pegeY 返回鼠标相对于文档页面Y坐标
          e.screenX 返回鼠标相对于电脑屏幕的X坐标
          e.screenY 返回鼠标相对于电脑屏幕的Y坐标
